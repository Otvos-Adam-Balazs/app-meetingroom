# App-meetingroom

This project is a **Spring Boot** application providing REST APIs.

## Requirements

- Java 21+ (or the version required by the project)  
- Maven  
- Docker and Docker Compose (if you want to run in a container)  

## Installation and Running


To build and run the project, use the following commands:

**Linux / macOS:**
```bash
./mvnw install
./mvnw spring-boot:run or docker compose up

**Windows:**
```bash
.\mvnw install
.\mvnw spring-boot:run or docker compose up


## User Test
The API can be tested via Swagger UI at: [http://localhost:8080/swagger-ui.html](http://localhost:8080/swagger-ui.html)
